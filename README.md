## 📏 측정하기
> - 측정에 필요한 데이터 입력 및 측정 진행  
> - **배관 타입:** Spinner 사용  
> - **관경:** 장비에서 측정된 데이터를 실시간 표시  
> - **Picture:** 카메라 앱 실행 및 촬영 (관 재질 촬영 목적)  
> - **초기화:** 모든 데이터 값 초기화  
> - **뒤로가기:** 이전 화면으로 이동  
> - **측정 결과 확인:** Fragment Popup에 표시 및 저장  
> - **저장 방식:** Room DB
